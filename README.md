## Getting Started
### Prerequisites
1. Bash Shell
### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/FerasHamam/Bash-Scripts.git
   ```
    
2. Give permissions to the script in Terminal
    ```sh
    chmod a+x z.sh
    ```
    z is the script name

3. Run the following in Terminal
   ```sh
   ./z.sh -h 
   ```
   z is the script name
